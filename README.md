## 🚀 Быстрый старт

### Вариант 1 — клонировать и запустить

```bash
git clone https://github.com/wrx861/bedolaga_auto_install.git
cd bedolaga_auto_install
chmod +x install.sh
./install.sh
```

### Вариант 2 — one-liner

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/wrx861/bedolaga_auto_install/main/install.sh)
```

> 💡 Установщик сам подготавливает сервер, проверяет окружение и открывает интерактивное меню.

## 📦 Режимы установки

### 1) Установить Бота
Ставит только **Bedolaga Bot**.

Подходит, если:
- у тебя уже есть Remnawave панель
- нужен только бот
- кабинет сейчас не нужен

### 2) Установить Кабинет
Ставит только **Bedolaga Cabinet**.

Подходит, если:
- бот уже есть отдельно
- нужен только frontend / cabinet слой

### 3) Установить Бот + Кабину
Ставит **оба компонента вместе**.

Подходит, если:
- хочешь собрать полный Bedolaga flow сразу
- нужен единый install scenario

## 🔄 Обновления

### Обновить Бота
Кнопка обновления бота выполняет:

```bash
git pull
docker compose down && docker compose up -d --build
```

### Обновить Кабину
Кнопка обновления кабинета выполняет:

```bash
git pull origin main
docker builder prune -a -f && docker compose down && docker compose build --no-cache && docker compose up -d
```

> ⚠️ Если во время обновления возникает ошибка, установщик не скрывает её и показывает пользователю сам текст ошибки.

## 🖥 Требования

Минимально:

- **Debian 11+** или **Ubuntu 20.04+**
- установленный `bash`
- доступ в интернет для установки зависимостей и клонирования репозиториев
- root / sudo-доступ на сервере

Рекомендуемо:

- **Debian 12**
- чистый VPS под тест или деплой
- корректно настроенный DNS, если планируется public-доступ
