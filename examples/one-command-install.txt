# One-command install from GitHub

## Recommended

bash <(curl -fsSL https://raw.githubusercontent.com/wrx861/bedolaga_auto_install/main/install.sh)

## Alternative

curl -fsSL https://raw.githubusercontent.com/wrx861/bedolaga_auto_install/main/install.sh | bash

## What happens
1. bootstrap installer repo into /opt/bedolaga-stack-installer
2. update existing installer there if already cloned
3. run the interactive Bedolaga installer menu
